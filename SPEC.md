# 睡眠診断【無料】あなたの睡眠の質をチェック・改善法を提案 - 技術仕様書

## 概要

**サービス名**: Sleep Improvement Checker
**バージョン**: 1.5.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/sleep-improvement-checker/

就寝時間・睡眠時間・起床状態から睡眠の問題を診断。不眠・睡眠負債の改善アドバイス付き。

## 更新履歴（v1.0.0 → v1.5.0）

**改善内容（2026-06-10 v1.4.0 → v1.5.0）**:
1. **M10: 印刷・結果エクスポート機能実装（P1）**: 結果画面に「印刷」「コピー」「CSV」の3つのボタンを追加。印刷ボタンで window.print() による印刷ダイアログを開く、コピーボタンで診断結果テキストをクリップボードにコピー、CSVボタンで日付入り CSV ファイルをダウンロード可能に。

**改善内容（2026-06-10 v1.3.0 → v1.4.0）**:
1. **M3: localStorage・履歴機能実装（P1）**: 診断結果を最大10件 localStorage に自動保存。前回スコアとの比較（改善度）を結果画面に表示。診断履歴一覧（details タグ）で過去の診断結果を確認・削除可能に。
2. **M4: スコア推移グラフ（Chart.js）実装（P2）**: 履歴が2件以上あれば、Chart.js で日付軸×スコア軸の折れ線グラフを結果画面に自動表示。モバイルでもレスポンシブに対応。

**改善内容（2026-06-03）**:
1. **モバイルレイアウト最適化**: スマートフォンでの選択肢を 2列グリッドに変更し、操作性を向上
2. **スコア表現の一貫性確保**: 利用者の声を修正し、「低いスコア＝睡眠の質が良い」という仕様を明確化
3. **スコア計算方法の可視化**: テーブル形式でスコア範囲と睡眠状態をマッピング、ユーザーが結果を正しく理解できるよう強化
4. **関連サービスの関連性向上**: 睡眠関連サービス（睡眠の質チェッカー、睡眠債務計算機等）に限定
5. **改善アドバイスの実用性向上**: 各推奨アクションに【今夜から】【1週間で】【継続】などの時間軸を明記し、段階的な改善を促進

**改善内容（2026-06-06 v1.2.0 → v1.3.0）**:
1. **プログレスバー逆転バグ修正（P1）**: 「低スコア＝良好」の仕様に合わせ、健康度バー `(30-total)/30*100%` に修正。スコア軸ラベル（0点=最良/15点/30点=最悪）を追加
2. **結果画面シェアボタン追加（P1）**: 判定結果・スコアを含むLINE・Xシェアボタンを結果カード内に配置し、スクロール不要でシェア可能に
3. **前の質問に戻る機能（P2）**: `answers[]` 配列でスコアを記録し、2問目以降に「← 前の質問に戻る」ボタンを表示。誤答時の修正が可能に
4. **スコア内訳表示（P2）**: 1点以上の項目を棒グラフ付きで表示し、最高スコア（最要注意）項目を「⚠️最優先」バッジで強調
5. **ローディング時間短縮（P2）**: 1800ms → 600ms に短縮。ローディング画面にドット点滅アニメーション追加

**改善内容（2026-06-04 v1.1.0 → v1.2.0）**:
1. **モバイルUX強化**: 選択肢ボタンのタッチターゲットを48px以上に確保、グリッド間隔を0.75remに拡大
2. **スコア視覚化改善**: 結果画面に0-30点の軸ラベルを表示し、スコアポジションを直感的に理解可能
3. **スコア内訳表示**: 各質問別の得点とテストで最も改善が必要な項目（最高スコア）をマーク表示
4. **結果シェア機能拡充**: LINE・X（Twitter）シェアボタンを結果画面上部に配置、テキスト形式で結果をダウンロード可能
5. **関連サービスの結果画面配置**: 睡眠の質チェッカー・睡眠債務計算機などを結果画面内に表示し、ユーザーの次アクション導線を強化

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## よくある質問（FAQ）

**Q: 睡眠診断は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
